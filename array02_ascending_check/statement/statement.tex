\section*{บันไดของมด}

ในป่าลึกแห่งหนึ่ง มีกลุ่มมดตัวจิ๋วที่ต้องเดินขึ้นบันไดที่สร้างจากหินก้อนเล็กๆ เรียงต่อกัน\\
โดยเจ้ามดหัวหน้าจะตรวจสอบว่าหินเหล่านี้เรียงจากต่ำไปสูงหรือไม่\\
(เช่น ระดับความสูงของแต่ละก้อนหินต้องเรียงจากน้อยไปมาก) เพื่อความปลอดภัยของมดตัวอื่นๆ
ที่จะเดินผ่าน

คุณในฐานะผู้ช่วยของมดหัวหน้า
ต้องช่วยเขียนโปรแกรมตรวจสอบว่าระดับความสูงของก้อนหินที่มดจะเดินผ่านเรียงจากน้อยไปมากหรือไม่


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรกเป็นจำนวนเต็มบวก\texttt{N}(1 ≤ N ≤ 100) แสดงจำนวนก้อนหิน
\item
  บรรทัดถัดไปเป็นจำนวนเต็ม\texttt{N}ค่า แสดงระดับความสูงของก้อนหินแต่ละก้อน
  (อาจซ้ำกันได้)
\end{itemize}


\subsection*{Output}
\begin{itemize}
\item
  หากระดับความสูงเรียงจากน้อยไปมากหรือเท่ากัน ให้พิมพ์\texttt{"safe"}
\item
  หากไม่เรียง ให้พิมพ์\texttt{"dangerous"}
\end{itemize}
