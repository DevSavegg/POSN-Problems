\section*{แสดงเลขคี่จากชุดข้อมูล}

ให้เขียนโปรแกรมเพื่อรับค่าจำนวนเต็ม\texttt{n}ตัวในบรรทัดเดียว (คั่นด้วยช่องว่าง)

แล้วให้แสดงเฉพาะเลขคี่ (Odd numbers) จากข้อมูลที่รับเข้ามา

โดยแสดงผลในบรรทัดเดียวกันและคั่นด้วยช่องว่าง หากไม่มีเลขคี่เลย
ให้แสดงคำว่า\texttt{none}\\


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม\texttt{n}(1 ≤ n ≤ 1000) แทนจำนวนตัวเลขที่ต้องรับ
\item
  บรรทัดที่สอง: จำนวนเต็ม\texttt{n}ตัว คั่นด้วยช่องว่าง
\end{itemize}


\subsection*{Output}
\begin{itemize}
\item
  แสดงเลขคี่ทั้งหมดในลำดับเดียวกัน คั่นด้วยช่องว่าง
\item
  หากไม่มีเลขคี่เลย ให้แสดงคำว่า\texttt{none}
\end{itemize}
