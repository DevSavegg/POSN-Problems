\section*{frequency}

นับความถี่ของตัวเลขในอาร์เรย์

คำอธิบาย:

- เขียนโปรแกรมรับข้อมูลตัวเลขจำนวน 10 ตัว เก็บในอาร์เรย์

- ตัวเลขที่รับต้องอยู่ในช่วง 0 ถึง 100 เท่านั้น

- นับความถี่ของแต่ละตัวเลขที่ปรากฏในอาร์เรย์

- แสดงผลลัพธ์เป็นตารางแสดงค่าและความถี่ของแต่ละตัวเลขที่พบ

- แสดงเฉพาะตัวเลขที่มีความถี่มากกว่า 0 เท่านั้น

- Pattern การเว้นวรรค ใช้ Tab 2 ครั้ง


\subsection*{Input}
{ตัวเลขที่รับต้องอยู่ในช่วง 0 ถึง 100 จำนวน{10 ตัว}}\\


\subsection*{Output}
{แสดงผลลัพธ์เป็นตารางแสดงค่าและความถี่ของแต่ละตัวเลขที่พบ}\\
