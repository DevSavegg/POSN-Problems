\section*{แสดงเลขกลับหลัง}

โปรแกรมจะรับจำนวนเต็ม N จำนวน จากนั้นรับเลข N ตัวในบรรทัดเดียวคั่นด้วยช่องว่าง

ให้แสดงเลขทั้งหมดในลำดับกลับหลัง โดยเว้นวรรคระหว่างตัวเลข


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม\texttt{N}(1 ≤ N ≤ 1000) --- จำนวนเลขที่จะรับ
\item
  บรรทัดที่สอง: จำนวนเต็ม N ตัว คั่นด้วยช่องว่าง
\end{itemize}


\subsection*{Output}
แสดงเลขทั้งหมดที่รับมาในลำดับกลับหลัง คั่นด้วยช่องว่างในบรรทัดเดียว\\
