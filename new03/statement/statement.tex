\section*{หาค่าเฉลี่ยเพื่อนในห้อง}

คุณครูต้องการหาค่าเฉลี่ยคะแนนสอบของนักเรียนในห้องโดยใช้ array

โปรแกรมจะรับจำนวนเต็ม N ซึ่งแทนจำนวนเพื่อนในห้อง

แล้วรับคะแนนสอบของเพื่อนแต่ละคน N ตัวในบรรทัดเดียว

ให้คำนวณและแสดงผลค่าเฉลี่ยของคะแนนเหล่านั้น โดยปัดเศษทศนิยม 2 ตำแหน่ง


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม N (1 ≤ N ≤ 100) --- จำนวนเพื่อนในห้อง
\item
  บรรทัดที่สอง: คะแนนสอบ N ตัว (จำนวนเต็ม 0--100) คั่นด้วยช่องว่าง
\end{itemize}


\subsection*{Output}
\begin{itemize}
\item
  แสดงค่าเฉลี่ยคะแนนสอบของเพื่อนในห้องโดยปัดเศษทศนิยม 2 ตำแหน่ง
\item
  ตัวอย่างเช่น 78.6666 ให้แสดง 78.67
\end{itemize}
