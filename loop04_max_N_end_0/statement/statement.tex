\section*{เครื่องชั่งเวทมนตร์ของพ่อมด ACS}

พ่อมด ``มาสเตอร์รูน'' กำลังทดลองวัตถุดิบเพื่อสร้างน้ำยาขยายพลัง\\
เขาจะชั่งน้ำหนักของวัตถุดิบแต่ละชิ้น และอยากรู้ว่าวัตถุดิบชิ้นใดหนักที่สุด

เขาจะป้อนน้ำหนักทีละชิ้น (รวมถึงวัตถุพิเศษที่มีน้ำหนักติดลบ)\\
เมื่อเจอเลข 0 หมายถึงหยุดชั่งทันที

\subsubsection{คำสั่ง:}\label{uxe04uxe33uxe2auxe07}

ให้เขียนโปรแกรมรับค่าน้ำหนักทีละค่า

เมื่อรับค่า 0 ให้หยุดรับ และแสดงค่าน้ำหนักที่มากที่สุดจากที่ชั่งทั้งหมด


\subsection*{Input}
{บรรทัดที่ 1 ถึง N รับจำนวนเต็มจนกว่าจะเจอ 0}\\


\subsection*{Output}
{แสดงน้ำหนักที่มากที่สุดที่ชั่งได้}\\
