\section*{ตัวกรองของหุ่นยนต์ทำความสะอาด}

ในโลกอนาคต หุ่นยนต์ทำความสะอาดชื่อว่า RoboClean-3
ทำงานด้วยระบบกรองสิ่งสกปรกอัจฉริยะ
มันสามารถแยกเศษขยะที่เหมาะสมกับระบบกำจัดขั้นสูงได้โดยใช้เกณฑ์พิเศษ:

\begin{quote}
"เศษที่นำเข้าสู่ระบบได้ ต้องมีค่าหารด้วย 3 ลงตัวและมีค่าเป็นเลขหลักเดียว (0-9) เท่านั้น"
\end{quote}

วิศวกรผู้ดูแล (ก็คือคุณ) ได้รับรายการรหัสขยะที่ RoboClean-3 ตรวจพบ
และต้องช่วยคัดกรองรายการให้เหลือเฉพาะรหัสที่หุ่นยนต์จะยอมรับเข้าสู่ระบบกำจัด


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม\texttt{n}(1 ≤ n ≤ 1000) --- จำนวนรหัสขยะ
\item
  บรรทัดที่สอง:\texttt{n}จำนวนเต็ม (ตั้งแต่\texttt{0}ขึ้นไป) ---
  รหัสขยะที่หุ่นยนต์ตรวจพบ
\end{itemize}


\subsection*{Output}
\begin{itemize}
\item
  แสดงรหัสขยะที่ผ่านเกณฑ์ (หาร 3 ลงตัว และเป็นเลขหลักเดียว) ทั้งหมดในบรรทัดเดียว
  เว้นวรรคคั่น
\item
  ถ้าไม่มีรหัสใดผ่านเกณฑ์เลย ให้แสดงคำว่า\texttt{None}
\end{itemize}
