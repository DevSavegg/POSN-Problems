\section*{พลังโจมตีของอัศวิน}

อัศวินแห่งอาณาจักรคูณเทีย (Multiplica) ต้องการสะสมพลังเพื่อเอาชนะมังกรไฟ
ด้วยการคูณพลังตั้งแต่ 1 ถึง N เข้าไว้ด้วยกัน

คุณในฐานะผู้ช่วยอัศวิน มีหน้าที่เขียนโปรแกรมคำนวณ "พลังโจมตี" ที่เท่ากับผลคูณของเลขตั้งแต่ 1
ถึง N (หรือที่เรียกว่า factorial)


\subsection*{Input}
บรรทัดเดียว รับจำนวนเต็มบวก\texttt{N}(1 ≤ N ≤ 50)\\


\subsection*{Output}
แสดงค่าผลคูณตั้งแต่ 1 ถึง N\\
