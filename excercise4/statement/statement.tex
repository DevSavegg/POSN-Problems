\section*{bus}

ป้ายรถเมล์แห่งหนึ่ง มีรถเมล์ทั้งหมด N คัน แต่ละคันจะไปในเส้นทางต่างกัน
และจะใช้เวลาในการวนกลับมาที่ป้ายรถเมล์แห่งนี้ต่างกัน (อาจมีสองคันใดๆที่ใช้เวลาเท่ากัน)
แต่สิ่งที่เหมือนกัน

คือ รถทั้ง N คันจะเริ่มออกตัวจากป้ายรถเมล์แห่งนี้พร้อมกัน การเดินรถเป็นวงวนเช่นนี้ทำให้คุณ!
ผู้โดยสารรถเมล์ขาประจำเกิดความสงสัยว่าอีกนานเท่าใด รถเมล์ทั้ง N คันจะวนมา ณ
ป้ายรถเมล์แห่งนี้พร้อมกันอีกครั้ง


\subsection*{Input}
ข้อมูลนำเข้าบรรทัดแรกรับจำนวนเต็ม N แทนจำนวนรถเมล์ทั้งหมด\\

บรรทัดที่สองรับจำนวนทั้งหมด N จำนวน คือ เวลาของรถแต่ละคันที่วนกลับมาที่ป้ายเดิม
(เวลามีหน่วยเป็นนาที)


\subsection*{Output}
{ระยะเวลาทั้งหมดเริ่มตั้งแต่รถเมล์ทั้ง N คันออกพร้อมกัน จนกระทั่งรถเมล์ทั้ง N คันวนกลับมา ณ
ป้ายรถเมล์ดังกล่าวพร้อมกันอีกครั้ง}\\
