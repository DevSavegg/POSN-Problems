\section*{หน้าเหมือนหลัง}

กำหนดข้อความ\texttt{s}มาให้ ให้หา\texttt{prefix}(คำข้างหน้า)
ที่\ul{\textbf{ยาวที่สุด}}ที่ตัวมันเองเป็น\texttt{suffix}(คำข้างท้าย) ด้วย

หากไม่มีคำเช่นนั้น ให้ตอบเป็นข้อความ\texttt{0}ตัวเดียว\\


\subsection*{Input}
บรรทัดแรก ข้อความ\texttt{s}{ความยาวไม่เกิน 5,000
ตัวอักษร}และมีแต่ตัวอักขระaถึงzตัวเล็กหมด\\


\subsection*{Output}
ข้อความ suffix ( ที่เท่ากับ suffix ) หรือ 0 ถ้าไม่มี\\
