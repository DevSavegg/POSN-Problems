\section*{ตู้เก็บของอัจฉริยะของคุณยาย}

คุณยายมีตู้เก็บของที่สามารถเรียกดูรายการของได้ผ่านแอปมือถือ
แต่วันหนึ่งคุณยายต้องการให้เอาของชิ้นแรกในลิสต์ไปวางไว้ท้ายสุด

และของท้ายสุดให้ย้ายมาไว้หน้าแทน เพื่อให้ง่ายต่อการหยิบของพิเศษเหล่านี้

คุณในฐานะหลานนักเขียนโปรแกรม
จะต้องช่วยเขียนโปรแกรมสลับของชิ้นแรกกับชิ้นสุดท้ายในลิสต์ให้คุณยายโดยเร็ว!


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม\texttt{n}(1 ≤ n ≤ 100) คือจำนวนของที่อยู่ในตู้
\item
  บรรทัดที่สอง: ตัวเลข\texttt{n}จำนวน (แต่ละตัวแทนชื่อของแบบรหัส เช่น 101 203 450
  เป็นต้น)
\end{itemize}


\subsection*{Output}
{แสดงรายการของใหม่หลังจากสลับของชิ้นแรกกับชิ้นสุดท้ายแล้ว โดยแสดงทั้งหมดบนบรรทัดเดียว
เว้นวรรคคั่นระหว่างตัวเลข}\\
