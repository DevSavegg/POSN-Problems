\section*{ภารกิจล่าจำนวนเฉพาะ}

โลกกำลังถูกคุกคามด้วยไวรัสดิจิทัลที่สามารถแฝงตัวอยู่ในเลขธรรมดา มีเพียง "จำนวนเฉพาะ"
เท่านั้นที่บริสุทธิ์และสามารถต่อต้านไวรัสได้

ภารกิจของคุณในฐานะโปรแกรมเมอร์ผู้กอบกู้โลก
คือการเขียนโปรแกรมเพื่อหาทุกจำนวนเฉพาะตั้งแต่ 1 ถึง N


\subsection*{Input}
บรรทัดเดียว รับจำนวนเต็มบวก\texttt{N}(1 ≤ N ≤ 100)\\


\subsection*{Output}
แสดงจำนวนเฉพาะทั้งหมดตั้งแต่ 1 ถึง N โดยเรียงจากน้อยไปมาก (เว้นวรรคคั่นแต่ละจำนวน)
