\section*{ละเลงสี}

ศิโรตม์มี กำแพง อยู่หลายแผ่นที่ต้องทาสี โดยแต่ละแผ่นมีกำหนด ค่าใช้จ่าย และ
เวลาที่ใช้ในการทาสี ไว้อยู่แล้ว

ศิโรตม์สามารถเลือกใช้ ช่างทาสี ได้สองแบบ:

1. ช่างทาสีที่ต้องจ่ายเงิน - สามารถทาสีกำแพงที่เลือกได้โดยใช้เวลาตามที่กำหนด
และต้องจ่ายเงินตามค่าที่กำหนด

2. ช่างทาสีฟรี - สามารถทาสีกำแพงให้เสร็จภายใน 1 หน่วยเวลา และไม่ต้องเสียเงิน
แต่จะสามารถทำงานได้ก็ต่อเมื่อ มีช่างทาสีที่ต้องจ่ายเงินกำลังทำงานอยู่

งานของคุณ: ช่วยศิโรตม์หาวิธีจ้างช่างทาสีเพื่อทาสีกำแพงทั้งหมดให้เสร็จด้วยค่าใช้จ่ายที่น้อยที่สุด

\textbf{ตัวอย่างที่ 1}

\textbf{Input:}

cost = {[}1,2,3,2{]}

time = {[}1,2,3,2{]}

\textbf{Output:}3

คำอธิบาย:

\begin{itemize}
\tightlist
\item
  ให้ช่างทาสีที่ต้องจ่ายเงินทาสีกำแพง หมายเลข 0 และ 1 ใช้เวลารวม 3 หน่วยเวลา
\item
  ระหว่างนั้น ให้ช่างทาสีฟรีช่วยทาสีกำแพง หมายเลข 2 และ 3 ซึ่งจะใช้เวลา 2 หน่วยเวลา
\item
  ค่าใช้จ่ายที่ต้องจ่ายทั้งหมดคือ 1 + 2 = 3
\end{itemize}

\textbf{ตัวอย่างที่ 2}

\textbf{Input:}

cost = {[}2,3,4,2{]}

time = {[}1,1,1,1{]}

\textbf{Output:}4

\textbf{คำอธิบาย:}

\begin{itemize}
\tightlist
\item
  ให้ช่างทาสีที่ต้องจ่ายเงินทาสีกำแพง หมายเลข 0 และ 3 ใช้เวลารวม 2 หน่วยเวลา
\item
  ระหว่างนั้น ให้ช่างทาสีฟรีช่วยทาสีกำแพง หมายเลข 1 และ 2 ซึ่งจะใช้เวลา 2 หน่วยเวลา
\item
  ค่าใช้จ่ายที่ต้องจ่ายทั้งหมดคือ 2 + 2 = 4
\end{itemize}

\textbf{เงื่อนไข:}

\begin{itemize}
\tightlist
\item
  จำนวนกำแพงมีอย่างน้อย 1 แผ่น และมากที่สุด 500 แผ่น
\item
  ค่าใช้จ่ายของแต่ละกำแพงอยู่ระหว่าง 1 ถึง 1,000,000
\item
  เวลาที่ใช้ทาสีแต่ละกำแพงอยู่ระหว่าง 1 ถึง 500 หน่วยเวลา
\end{itemize}


\subsection*{Input}
บรรทัดแรก: ตัวเลขจำนวนเต็ม\texttt{N}แทนจำนวนกำแพงทั้งหมด

บรรทัดที่สอง:{ตัวเลขจำนวนเต็มแทน}ค่าใช้จ่ายในการทาสีกำแพงช่องนั้น ๆ คั่นด้วยช่องว่าง

บรรทัดที่สาม:{ตัวเลขจำนวนเต็มแทนเวลาที่ต้องใช้ในการทาสีกำแพงช่องนั้น ๆคั่นด้วยช่องว่าง}


\subsection*{Output}
จำนวนเต็มแทนค่าใช้จ่าย{ที่ต้องจ่ายทั้งหมด}ที่น้อยที่สุด
