\section*{คำถี่สูงสุด Q คำ}

กำหนดข้อมูลเข้าเป็นข้อความ (string) ที่เริ่มต้นด้วยจำนวนเต็ม Q
แล้วตามด้วยคำภาษาอังกฤษต่างๆ จำนวนมากกว่า Q คำ เว้นด้วยช่องว่าง
(space)\ul{\textbf{ให้หาคำในข้อความนั้นที่มีความถี่สูงสุดตัว Q
ตัวแรก}}ผลลัพธ์ให้แสดงคำนั้นและตามด้วยจำนวนครั้งที่พบด้วย
และ\ul{\textbf{ให้เรียงคำที่มีความถี่เท่ากันตามตัวอักษร (lexicographical order)
ทั้งนี้คำต่างๆ เป็นตัวอักษรเล็กหมด}}.\\


\subsection*{Input}
คำในข้อความนั้นที่มีความถี่สูงสุดตัว Q ตัวแรก\\


\subsection*{Output}
{คำในข้อความนั้นที่มีความถี่สูงสุดตัว Q ตัวแรก}ให้เรียงคำที่มีความถี่เท่ากันตามตัวอักษร (
lexicographical order ) เป็นตัวอักษรเล็กหมด{.}\\
