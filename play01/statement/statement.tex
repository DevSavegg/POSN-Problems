\section*{เสียงเชียร์จากแฟนคลับ}

ในรายการแข่งขันร้องเพลงชื่อดังระดับประเทศอย่าง "\textbf{The Voice of Algorithm}"
ผู้เข้าแข่งขันแต่ละคนจะมีรหัสประจำตัว (เป็นเลขตั้งแต่ 1 ขึ้นไป)\\
ระหว่างแข่งขัน แฟนคลับจะกดเชียร์รหัสของผู้เข้าแข่งขันที่พวกเขาชื่นชอบ

ทีมงานต้องการหาว่า\textbf{ใครคือผู้เข้าแข่งขันที่ได้รับเสียงเชียร์ ``มากที่สุด'' จากแฟนคลับ}

แต่ในกรณีที่มีผู้เข้าแข่งขันหลายคนได้เสียงเท่ากัน
ให้แสดง\textbf{หมายเลขของผู้เข้าแข่งขันทั้งหมดเรียงจากน้อยไปมาก}


\subsection*{Input}
\begin{itemize}
\item
  บรรทัดแรก: จำนวนเต็ม\texttt{N}(1 ≤ N ≤ 100) --- จำนวนเสียงเชียร์ทั้งหมด
\item
  บรรทัดที่สอง: มี\texttt{N}ตัวเลข เป็นรหัสของผู้เข้าแข่งขันที่ได้รับเสียงเชียร์
\end{itemize}


\subsection*{Output}
\begin{itemize}
\item
  แสดงรหัสของผู้เข้าแข่งขันที่ได้รับเสียงเชียร์มากที่สุด\textbf{เรียงจากน้อยไปมาก}คั่นด้วยช่องว่าง
\item
  ถ้าไม่มีใครได้เสียงซ้ำ (เชียร์แค่คนละ 1 ครั้ง) ให้แสดง\texttt{"None"}
\end{itemize}
