\section*{รูปปั้นพิฆาต}

แอนดี้ และมาร์คได้หลุดเข้าไปยังต่างโลกที่กำหนดให้เขาทั้ง 2
ต้องเล่นเกมเพื่อเอาชีวิตรอดซึ่งคู่แข่งในเกมนี้ก็คือรูปปั้นต่างๆ
โดยเกมนี้กำหนดให้เล่นบนกระดานหมากรุกขนาด 8x8
ซึ่งแอนดี้ถูกกำหนดให้อยู่ที่มุมบนขวาของกระดาน และมาร์คอยู่ที่มุมล่างซ้ายของกระดาน
นอกจากแอนดี้และมาร์คแล้ว บนกระดานยังมีรูปปั้นอื่น ๆ
อีกหลายตัวซึ่งแต่ละตัวจะอยู่ในช่องหนึ่งช่องโดยที่ช่องที่มีรูปปั้นจะไม่สามารถมีสิ่งอื่นใดอยู่ได้
ไม่ว่าจะเป็นรูปปั้นตัวอื่น, แอนดี้ หรือ มาร์ค

แอนดี้จะยืนอยู่ที่ตำแหน่งของตัวเองและไม่ขยับ
แต่มาร์คจะมีการเคลื่อนไหวโดยเป้าหมายของมาร์คคือการเดินไปหาตำแหน่งของแอนดี้
มาร์คและรูปปั้นจะเคลื่อนไหวสลับกันโดยที่มาร์คจะเริ่มเดินก่อน ในแต่ละตา
มาร์คจะสามารถเดินไปยังช่องที่ติดกับช่องที่ตัวเองอยู่ได้ (ทั้งในแนวตั้ง แนวนอน หรือในแนวทแยง)
แต่ช่องนั้นจะต้องไม่มีรูปปั้นหรือจะเลือกที่จะยืนอยู่ที่ตำแหน่งเดิมก็ได้
ส่วนรูปปั้นจะเคลื่อนที่ลงไปหนึ่งช่องในแต่ละตาและรูปปั้นที่อยู่ในแถวล่างสุดจะตกออกจากกระดานไป

หากรูปปั้นไปอยู่ในช่องที่มาร์คอยู่ รูปปั้นจะชนะ แต่ถ้ามาร์คเดินไปถึงช่องที่แอนดี้ยืนอยู่มาร์คจะชนะ
เกมจะจบเมื่อมีฝ่ายหนึ่งฝ่ายใดชนะ โดยไม่เกี่ยวข้องกับจำนวนรูปปั้น


\subsection*{Input}
คุณจะได้รับข้อมูลที่เป็นกระดานหมากรุก 8 แถว โดยแต่ละแถวมี8ช่อง ซึ่งแถวแรกคือแถวบนสุด
และแถวสุดท้ายคือล่างสุด

\begin{itemize}
\tightlist
\item
  ถ้าช่องว่างจะมีสัญลักษณ์เป็น "."
\item
  ถ้ามีมาร์คอยู่ในช่องนั้นจะเป็น "M"
\item
  ถ้ามีแอนดี้อยู่ในช่องนั้นจะเป็น "A"
\item
  ถ้ามีรูปปั้นอยู่ในช่องนั้นจะเป็น "S"
\end{itemize}

เงื่อนไข:

\begin{itemize}
\tightlist
\item
  ตัว "A" จะอยู่ที่ช่องสุดท้ายในแถวแรก
\item
  ตัว "M" จะอยู่ที่ช่องแรกในแถวสุดท้าย
\item
  ช่องที่เหลือจะเป็น "." หรือ "S"
\end{itemize}


\subsection*{Output}
\begin{itemize}
\tightlist
\item
  ถ้ามาเรียชนะให้พิมพ์คำว่า "WIN"
\item
  ถ้ารูปปั้นชนะให้พิมพ์คำว่า "LOSE"
\end{itemize}
